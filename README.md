# TheBalaBot / Nocthera

A beginner-friendly Discord bot for the Nocthera community.

## Commands
- `/ping`
- `/tarot`
- `/cat`
- `/help`

## Railway variables
Add these in Railway → Variables:

- `DISCORD_TOKEN`
- `CLIENT_ID`
- `WELCOME_CHANNEL`

## Start command
Railway should use:

```bash
npm start
```

## Register slash commands
In Railway or locally, run:

```bash
npm run deploy-commands
```
